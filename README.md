

#   PetHeaven

    A full-stack pet adoption platform that connects loving families with pets in need. Every adoption saves a life and creates a beautiful bond.



#   Live URL
    [https://petheaven-seven.vercel.app](https://petheaven-seven.vercel.app)



#   Purpose

    PetHeaven is a web application designed to simplify and humanize the pet adoption process. It allows users to browse available pets, submit adoption requests, and manage their listings — all from a clean, modern interface. The platform also supports donation campaigns and provides pet care tips and advice to help new owners give their pets the best life possible.


#   Features

    *   Browse & Search Pets** — Explore a wide variety of available pets (dogs, cats, birds, rabbits, and more) with detailed profiles including breed,
        age, gender, and location.
    *   Adopt Now System — Logged-in users can send adoption requests directly from a pet's detail page and track the status of their requests from 
        the dashboard.
    *   User Dashboard — A personalized dashboard where users can manage their own pet listings, view incoming adoption requests, and track their own 
        adoption applications.
    *   Add a Pet for Adoption — Registered users can list their own pets for adoption by providing photos, breed info, and other details.
    *   Donation Campaigns — Users can donate to support rescue shelters and ongoing animal welfare campaigns through an integrated donation system.
    *   Pet Care Blog — An informative blog section covering pet nutrition, grooming, and behavior tips written by experts.
    *   Authentication System — Secure user authentication using `better-auth`, supporting sign-up, login, and session management.
    *   Responsive Design — Fully responsive UI built with Tailwind CSS and HeroUI, providing a seamless experience on both desktop and mobile.
